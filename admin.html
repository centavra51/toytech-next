Готовое подключение Google Sheets

Данные будут сохраняться в вашей Google Таблице И будут доступны с любого устройства. При передаче клиенту — он просто вставляет свою ссылку в это поле.

Apps Script URL
Инструкция: как подключить Google Sheets
  1. Открой sheets.new — создай таблицу «ToyTech CMS»
  2. В таблице: верхнее меню → Extensions → Apps Script
  3. Удали старый код, вставь код из файла apps-script.gs
  4. Нажми Deploy → New deployment → Web App
  5. Установи: «Execute as: Me» и «Who has access: Anyone»
  6. Нажми Deploy → скопируй URL → вставь выше
